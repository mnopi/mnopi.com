|---|---|

| **formula** | **0**, 1 | [repos](https://docs.github.com/en/rest/reference/repos#get-a-repository--code-samples) | Generate values for homebrew formula |

| **latest** | **0**, 1 | [repos](https://docs.github.com/en/rest/reference/repos#get-a-repository--code-samples) | Latest archive releases or tag url |

### [repos](https://mnopi.com/api/git?index=repos)

#### formula (uses releases)
```bash
curl -fsSL "https://mnopi.com/api/git?formula=1"
```

#### formula for python/cpython (uses tag)
```bash
curl -fsSL "https://mnopi.com/api/git?formula=1&owner=python&repo=cpython"
```


### [search](https://mnopi.com/api/git?index=search)
```bash
curl -fsSL "https://mnopi.com/api/git"
curl -fsSL "https://mnopi.com/api/git?formula=1"
curl -fsSL "https://mnopi.com/api/git?index=search&q=homebrew-+user:homebrew"
```

#### taps
```bash
curl -fsSL "https://mnopi.com/api/git"
curl -fsSL "https://mnopi.com/api/git?log=1"
curl -fsSL "https://mnopi.com/api/git?index=search&q=homebrew-+user:homebrew"
```

#### repos with "cmd", in user "j5pu"
```bash
curl -fsSL "https://mnopi.com/api/git?index=search&q=cmd+user:j5pu"
```

#### repos starting with "cmd", in user "j5pu"
```bash
curl -fsSL "https://mnopi.com/api/git?index=search&q=^cmd+user:j5pu"
```
